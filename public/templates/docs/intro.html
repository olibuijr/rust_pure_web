Zero-Dependency Philosophy

No tokio. No hyper. No serde. Everything built with std only. Custom implementations of SHA-256, PBKDF2, ChaCha20, HTTP parsing, template engine, and more.

Quick Start

# Prerequisites
# Rust 1.70+ required (rustup.rs)

# Clone and build
git clone https://github.com/olibuijr/rust_pure_web.git
cd rust_pure_web
cargo build --release

# Configure environment (optional)
cp .env.example .env.local
# Edit .env.local with your settings

# Run the server
./target/release/rust_pure_web

# Visit in your browser
http://localhost:3460        # Main application
http://localhost:3460/_admin # Admin panel
http://localhost:3460/docs   # Documentation

First-time setup creates a default admin user. See Authentication for details on user management.

Configuration

# .env.local
SECRET_KEY="your-256-bit-secret-key"
ADMIN_EMAIL="admin@example.com"
ADMIN_PASSWORD="secure-password"

Default admin is created on first startup if no users exist.
