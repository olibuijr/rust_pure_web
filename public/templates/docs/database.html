Database

In-memory document store with encrypted persistence.

In-Memory ChaCha20 Encrypted Binary Format

Overview

The database is a document store similar to MongoDB or PocketBase. Data is kept in memory for fast access and automatically synced to an encrypted binary file. Collections can be created dynamically via the API or admin panel.

Data Types

The database supports these value types:

enum Value {
    Null,
    Bool(bool),
    Int(i64),
    Float(f64),
    String(String),
    Array(Vec<Value>),
    Object(HashMap<String, Value>),
}

Auto-Generated Fields

Every document automatically gets these fields:

id Random 24-character hex string
created Unix timestamp (seconds) when document was created
updated Unix timestamp (seconds) of last update
