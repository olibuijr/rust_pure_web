Tailwind CSS 4 with dark mode, custom theme, and animated backgrounds.

Overview

Styling is built on Tailwind CSS 4 with a custom dark theme using HSL color variables. The design features an animated "lava lamp" background effect and follows the shadcn/ui color system.

Tailwind CSS 4 Setup

The input.css File

Located at public/input.css, this is your Tailwind source file.

@import "tailwindcss";

@source "./public/**/*.html";

@theme {
  --color-border: hsl(240 3.7% 15.9%);
  --color-input: hsl(240 3.7% 15.9%);
  --color-ring: hsl(240 4.9% 83.9%);
  --color-background: hsl(240 10% 3.9%);
  --color-foreground: hsl(0 0% 98%);
  --color-primary: hsl(0 0% 98%);
  --color-primary-foreground: hsl(240 5.9% 10%);
  --color-secondary: hsl(240 3.7% 15.9%);
  --color-secondary-foreground: hsl(0 0% 98%);
  --color-muted: hsl(240 3.7% 15.9%);
  --color-muted-foreground: hsl(240 5% 64.9%);
  --color-accent: hsl(240 3.7% 15.9%);
  --color-accent-foreground: hsl(0 0% 98%);
  --color-card: hsl(240 10% 3.9%);
  --color-card-foreground: hsl(0 0% 98%);
}

Compiling CSS

Run the Tailwind CLI to compile your styles:

# One-time build
npx tailwindcss -i public/input.css -o public/styles.css

# Watch mode during development
npx tailwindcss -i public/input.css -o public/styles.css --watch

# Minified production build
npx tailwindcss -i public/input.css -o public/styles.css --minify

The Animated Background

The signature "lava lamp" effect uses layered radial gradients with a slow CSS animation.

Base Layer (body)

Static layered gradients provide the base effect.

@layer base {
  body {
    @apply bg-background text-foreground;
    background-image:
      radial-gradient(1200px 700px at 20% -10%,
        rgba(255, 255, 255, 0.18), transparent 60%),
      radial-gradient(1000px 600px at 80% 0%,
        rgba(255, 255, 255, 0.14), transparent 55%),
      radial-gradient(900px 700px at 50% 120%,
        rgba(255, 255, 255, 0.12), transparent 60%),
      linear-gradient(180deg,
        rgba(15, 23, 42, 0.55), rgba(2, 6, 23, 0.7));
    background-size: 120% 120%, 140% 140%, 160% 160%, 100% 100%;
    background-attachment: fixed;
  }
}

Animated Layer (body::before)

A pseudo-element adds the slow "lava" movement.

body::before {
  content: "";
  position: fixed;
  inset: -20%;        /* Extends beyond viewport */
  z-index: -1;
  background:
    radial-gradient(800px 600px at 20% 0%,
      rgba(255, 255, 255, 0.12), transparent 60%),
    radial-gradient(700px 500px at 80% 10%,
      rgba(255, 255, 255, 0.1), transparent 60%),
    radial-gradient(900px 700px at 50% 100%,
      rgba(255, 255, 255, 0.08), transparent 60%);
  filter: blur(20px); /* Soft edges */
  transform: translateY(-10%);
  animation: lava 36s ease-in-out infinite;
}

@keyframes lava {
  0%   { transform: translateY(-10%); }
  50%  { transform: translateY(12%); }
  100% { transform: translateY(-10%); }
}

View Transitions

CSS view transitions provide smooth page navigation.

<!-- components/styles.html -->
<style>
  @view-transition { navigation: auto; }

  /* Named transition elements */
  .docs-nav { view-transition-name: docs-nav; }
  .docs-sidebar { view-transition-name: docs-sidebar; }
  .docs-content { view-transition-name: docs-content; }

  /* Keep nav/sidebar static during transitions */
  ::view-transition-old(docs-nav),
  ::view-transition-new(docs-nav),
  ::view-transition-old(docs-sidebar),
  ::view-transition-new(docs-sidebar) {
    animation: none;
  }

  /* Content fades */
  ::view-transition-old(docs-content) {
    animation: fade-out 0.2s ease-out;
  }
  ::view-transition-new(docs-content) {
    animation: fade-in 0.2s ease-in;
  }
</style>

Content Entry Animation

The ui-runtime.html component animates content on load.

<!-- components/ui-runtime.html -->
<style>
  .content-enter {
    opacity: 0;
    transform: translateY(8px);
    transition: opacity 260ms ease, transform 260ms ease;
  }
  .content-enter.content-ready {
    opacity: 1;
    transform: translateY(0);
  }
</style>
<script>
  window.addEventListener('DOMContentLoaded', () => {
    document.querySelectorAll('[data-content-animate]').forEach((el) => {
      el.classList.add('content-ready');
    });
  });
</script>

<!-- Usage in templates -->
<main class="content-enter" data-content-animate>
  <!-- Content animates in -->
</main>

<div class="px-4 md:px-6 lg:px-8">

Using the Theme Colors

Text Colors

<!-- Primary text (white in dark mode) -->
<p class="text-foreground">Main text</p>

<!-- Muted text (gray) -->
<p class="text-muted-foreground">Secondary text</p>

<!-- Button text on primary background -->
<button class="bg-primary text-primary-foreground">

Background Colors

<!-- Main background -->
<body class="bg-background">

<!-- Card/surface -->
<div class="bg-card">

<!-- Secondary surface -->
<div class="bg-secondary">

<!-- With transparency -->
<div class="bg-secondary/50">

Interactive States

<!-- Button hover -->
<button class="bg-primary hover:bg-primary/90 transition-colors">

<!-- Link hover -->
<a class="text-muted-foreground hover:text-foreground transition-colors">

<!-- Card hover -->
<div class="bg-card hover:bg-secondary/50 transition-colors">

  • Dark mode: Always on via class="dark" on html
  • Background: Animated lava effect in input.css @layer base
