Installation

Get Rust Pure Web up and running in minutes.

Rust 1.70+ No Dependencies 5 min setup

Prerequisites

You need Rust 1.70 or later installed on your system. If you don't have Rust yet, install it via rustup: 

curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh

Verify your installation with rustc --version

Step 1: Clone the Repository

git clone https://github.com/olibuijr/rust_pure_web.git
cd rust_pure_web

Step 2: Build the Project

Build in release mode for optimal performance: 

cargo build --release

This compiles the entire framework with zero external dependencies. The binary will be at ./target/release/rust_pure_web

Step 3: Configure Environment

Create a .env.local file in the project root with your configuration: 

# .env.local
SECRET_KEY="your-256-bit-secret-key-here"
ADMIN_EMAIL="admin@example.com"
ADMIN_PASSWORD="your-secure-admin-password"
SECRET_KEY Used for ChaCha20 database encryption and session tokens. Use a random 32+ character string.
ADMIN_EMAIL Email for the default admin account created on first startup.
ADMIN_PASSWORD Password for the admin account. Hashed with PBKDF2 before storage.

Step 4: Run the Server

Start the server using either method: 

# Development (with hot reload)
cargo run

# Production (optimized binary)
./target/release/rust_pure_web

Step 5: Verify Installation

Open your browser and visit these URLs:

http://localhost:3460 Main application
http://localhost:3460/_admin Admin panel
http://localhost:3460/docs Documentation

Optional: Tailwind CSS Setup

If you want to customize styles using Tailwind CSS, install the dependencies and run the watcher: 

# Install dependencies
npm install

# Watch for changes and rebuild CSS
npx tailwindcss -i ./src/input.css -o ./public/css/styles.css --watch

Run this in a separate terminal alongside the Rust server.

Installation Complete

You now have a fully functional web framework with authentication, database, admin panel, and hot reload. Continue to the Quick Start guide to create your first page.
