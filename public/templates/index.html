Rust Pure Web

A zero-dependency web framework built entirely with Rust's standard library. No npm. No external crates. Just pure Rust.

0 dependencies
~1,800 lines of code
Sub-second builds
Documentation Source

This project was born from a desire to strip away the complexity that has crept into modern web development and return to the fundamentals - building everything from first principles using only what Rust's standard library provides. It represents my philosophy that simpler is better, and that truly understanding your tools leads to better software.

What's Included

🔐

Authentication

  • • Session-based auth with secure tokens
  • • PBKDF2 password hashing (100k iterations)
  • • Role-based access control
🗃

Database

  • • In-memory document store
  • • ChaCha20 encryption at rest
  • • Auto-generated timestamps
⚡

REST API

  • • Auto-generated CRUD endpoints

  • • Built-in auth middleware

Admin Panel

Templates

Hot Reload
